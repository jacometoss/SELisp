SELisp

Estas aplicaciones están estructuradas en módulos especializados desarrollados para la plataforma AutoCAD / DraftSight, siguiendo el estándar de codificación vigente, con el objetivo de realizar el dimensionamiento de conductores eléctricos y generar el desglose de la memoria descriptiva correspondiente a instalaciones eléctricas.

Tradicionalmente, estos módulos solo podían cargarse y ejecutarse en versiones completas de Autodesk AutoCAD, así como en DraftSight o BricsCAD. No obstante, actualmente AutoCAD LT 2026 incorpora compatibilidad que permite ejecutar estas aplicaciones, ampliando así las capacidades funcionales de esta versión.

SELisp, como su nombre lo indica, corresponde a un conjunto de módulos orientados a Sistemas Eléctricos de baja tensión (SE). El término LISP hace referencia al lenguaje de programación utilizado, cuyo nombre proviene del concepto de procesamiento de listas.

SUITE SELisp

SUITE SELisp es un conjunto de aplicaciones para AutoCAD, diseñado para el dimensionamiento de conductores eléctricos de cobre y aluminio en sistemas de baja tensión de corriente alterna. Integra módulos especializados que permiten realizar cálculos precisos y eficientes directamente sobre planos eléctricos. Es ideal para ingenieros, arquitectos, diseñadores y proyectistas eléctricos. Disponible con licencia digital válida por 6 meses o 1 año, según tus necesidades.

Más información en: https://selisp.com/suite

Comandos

Los comandos disponibles de las aplicaciones para no entrar en interferencia con los programas son los siguientes :

  • vd01 - Iniciar la aplicación MODBTCU. (Disponible)
  • vd02 - Iniciar la aplicación MODBTAL. (Disponible)
  • vd03 - Iniciar la aplicación MODBTCU.
  • vd04 - Iniciar la aplicación MODBTAL.
  • pipeline - Iniciar la aplicación ADUCT. (Disponible)
  • fv01 - Iniciar la aplicación MODBTPV01.
  • fv02 - Iniciar la aplicación MODBTPV02.
